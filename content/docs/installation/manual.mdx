<Steps>

    <Step> Add Tailwind CSS</Step>

    The components are styled with Tailwind CSS, so you'll need to install Tailwind CSS in your project.

    Follow the [Tailwind CSS installation](https://tailwindcss.com/docs/installation) guide to get started.

    <Step> Add dependencies</Step>

    Add these dependencies to your project:

    <InstallationCommand command="npm install tailwind-variants clsx tailwind-merge lucide-react" />

    <Step>Add tailwind color config to `@theme` layer</Step>

    Here's app.css file looks like:

    <Step>Edit tsconfig.json file (If your project in typescript)</Step>

    Add the following code to the `tsconfig.app.json` file to resolve paths for your IDE:

    <CodePreview lang="js" code={`{
    "compilerOptions": {
        // ...
        "baseUrl": ".", // [!code highlight]
        "paths": { // [!code highlight]
            "@/*": [ // [!code highlight]
                "./src/*" // [!code highlight]
            ] // [!code highlight]
        } // [!code highlight]
    // ...
    }
}`} />

    The `@` alias is just a preference. Feel free to use any other alias you prefer.

    <Step>Add a `cn` utility function</Step>

    <CodePreview lang="js" code={`import { clsx, type ClassValue } from "clsx"

import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs))
}`} />

    <Step>That's it</Step>

    You can now start adding components to your project.

</Steps>
