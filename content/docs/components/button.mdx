## Props

| Prop     | Type                                    | Default |
| -------- | --------------------------------------- | ------- |
| variant  | `solid` `outline` `soft` `ghost`        | solid   |
| size     | `sm` `md` `lg`                          | md      |
| color    | `default` `primary` `secondary` `error` | default |
| onlyIcon | Boolean                                 | false   |
