## Props

### Select

| Prop    | Type                                    | Default |
| ------- | --------------------------------------- | ------- |
| variant | `outline` `soft` `underline`            | outline |
| color   | `default` `primary` `secondary` `error` | default |
| size    | `sm` `md` `lg`                          | md      |

### SelectTrigger

| Prop | Type      | Default               |
| ---- | --------- | --------------------- |
| icon | ReactNode | `<ChevronDownIcon />` |
