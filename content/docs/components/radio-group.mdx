## Props

| Prop    | Type                                              | Default |
|---------|---------------------------------------------------|---------|
| variant | `soft` `outline`                                  | outline |
| color   | `default` `primary` `secondary` `success` `error` | default |
| size    | `sm` `md` `lg `                                   | md      |
