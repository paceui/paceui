## Props

| Prop    | Type                                    | Description           | Default |
| ------- | --------------------------------------- | --------------------- | ------- |
| variant | `solid` `soft`                          | To change the variant | solid   |
| color   | `default` `primary` `secondary` `error` | To change the color   | default |
| size    | `sm` `md` `lg `                         | To change the size    | md      |
