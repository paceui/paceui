## Props

| Prop  | Type                                    | Default |
| ----- | --------------------------------------- | ------- |
| shape | `round` `square`                        | round   |
| size  | `sm` `md` `lg`                          | md      |
| color | `default` `primary` `secondary` `error` | default |
