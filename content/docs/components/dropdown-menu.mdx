## Props

### DropdownMenuItem

Use same props to: `<DropdownMenuSubTrigger />` `<DropdownMenuCheckboxItem />` `<DropdownMenuRadioItem />`

| Prop    | Type                                              | Default |
|---------|---------------------------------------------------|---------|
| variant | `solid` `outline` `soft` `ghost`                  | soft    |
| color   | `default` `primary` `secondary` `success` `error` | default |
