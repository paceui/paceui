# PaceUI

A collection of ready-to-use component variants that you can easily copy, paste, and customize for React and Next.js apps.
Stay tuned for more components, snippets, complete blocks, and design resources coming your way!

## Getting Started

Head over to the [paceui.com](https://paceui.com/) documentation to get started!
